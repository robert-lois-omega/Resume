APM: Automatic Printing Machine V4

Version 4 of the existing technology caters to wireless printing services to the public and automatically deletes the files promoting the data flow's security. The users can upload through their desired location where they can pick up their order. The machine automatically computes the price for both colored and black-and-white attributes of every page and calculates the actual price before the prices set by the owner of the machine to be paid by the user through Gcash. The prototype does not use the API to check if the order was paid, instead, it uses OCR to scan the values of the online receipt that was uploaded as well as uses AI to validate the receipt layout and fonts as a countermeasure to fraud.
